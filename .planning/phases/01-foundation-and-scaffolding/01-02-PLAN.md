**db/models/ (all model files)**: Implement ALL models from `docs/design/DATA_MODELS.md` using SQLAlchemy 2.0 Mapped columns:
- **project.py**: Project (with all ProjectStatus and ProjectType enums as Python enums), Pipeline (PipelineStatus enum), PipelinePhase (PhaseType, PhaseStatus enums). Foreign keys: Pipeline.project_id -> Project.id, PipelinePhase.pipeline_id -> Pipeline.id.
- **agent.py**: Agent (AgentType, AgentStatus enums), AgentExecution (ExecutionStatus enum). FK: Agent.project_id -> Project.id, AgentExecution.agent_id -> Agent.id, AgentExecution.task_id -> Task.id.
- **task.py**: Task (TaskStatus enum). FK: Task.project_id -> Project.id, Task.phase_id -> PipelinePhase.id, Task.parent_task_id -> Task.id (self-referential).
- **artifact.py**: CodeArtifact. FK: CodeArtifact.project_id -> Project.id, CodeArtifact.agent_id -> Agent.id.
- **test_result.py**: TestResult (TestStatus enum). FK: TestResult.project_id -> Project.id.
- **security.py**: SecurityFinding (FindingType, Severity, FindingStatus enums). FK: SecurityFinding.project_id -> Project.id.
- **review.py**: ReviewComment (CommentType, CommentStatus enums, reuses Severity). FK: ReviewComment.project_id -> Project.id, ReviewComment.agent_id -> Agent.id.
- **event.py**: Event (EventType enum). FK: Event.project_id -> Project.id.
- **checkpoint.py**: Checkpoint with pipeline_id, node_id, state_data (JSON), created_at. FK: Checkpoint.pipeline_id -> Pipeline.id.
- **experiment.py**: ExperimentLog (ExperimentStatus enum). FK: ExperimentLog.project_id -> Project.id, ExperimentLog.pipeline_id -> Pipeline.id.
- **user.py**: User (id, email unique, name, hashed_password, is_active, created_at), ApiKey (id, user_id FK, key_hash, name, scopes JSON, created_at, expires_at, is_active), AuditLog (id, user_id FK, action, resource_type, resource_id, details JSON, ip_address, created_at).

Use Python `enum.Enum` for all status/type enums. Store enums in the database using `sa.Enum(EnumClass, name="enum_name", create_constraint=True)`. Use `Mapped[uuid.UUID]` with `mapped_column(sa.Uuid, primary_key=True, default=uuid.uuid4)` for all ID columns.
