<objective>
Build the SOC 2 compliance subsystem: a compliance checker that evaluates generated code
against Trust Service Criteria, an immutable audit logger with content hashing, and an
evidence collector for auditor export.

Purpose: Enables CodeBot to provide SOC 2 Type II readiness for both the platform itself
(immutable audit trails) and generated code (TSC compliance checking). The compliance
checker integrates into the SecurityOrchestrator as an optional 4th parallel scanner.

Output: Compliance checking package with TSC evaluation, audit logger, evidence exporter,
YAML configuration, and unit tests.
</objective>
