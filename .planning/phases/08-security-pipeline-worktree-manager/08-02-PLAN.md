Build the worktree pool manager, dynamic port allocator, branch/merge strategy, and commit manager that provide isolated working directories for parallel coding agents.

Output: Worktree management package with pool, port allocator, branch strategy, commit manager, and unit tests.

    3. Create `apps/server/src/codebot/worktree/pool.py` with `WorktreePool` class:

       Constructor: `__init__(self, repo_path: str, pool_dir: str, pool_size: int = 5)`. Store repo_path, pool_dir, pool_size. Create `self.available: asyncio.Queue[WorktreeInfo] = asyncio.Queue()`, `self.active: dict[str, WorktreeInfo] = {}`, `self._lock = asyncio.Lock()`, `self._initialized = False`.

       `async def initialize(self) -> None`: Run `git worktree prune` on repo_path (via _run_git). Create pool_dir directory (os.makedirs with exist_ok=True). For i in range(pool_size): call `await self._create_worktree(f"pool-{i}")` and `await self.available.put(wt)`. Set `self._initialized = True`.

       `async def acquire(self, agent_id: str, branch_name: str) -> WorktreeInfo`: Try `self.available.get_nowait()`. On `asyncio.QueueEmpty`: call `self._create_worktree(f"overflow-{uuid4().hex[:8]}", is_overflow=True)`. Then run `git checkout -b {branch_name}` in worktree path via _run_git. Set wt.branch=branch_name, wt.agent_id=agent_id. Under self._lock: `self.active[wt.id] = wt`. Return wt.

       `async def release(self, worktree: WorktreeInfo) -> None`: Under self._lock: pop from active. Run `git checkout main` then `git clean -fd` then `git reset --hard HEAD` in worktree path. If `self.available.qsize() < self.pool_size` and `not worktree.is_overflow`: put back in available. Else: `await self._destroy_worktree(worktree)`.

       `async def cleanup(self) -> None`: Drain available queue destroying each. Destroy all active under lock. Run `git worktree prune` on repo_path.

       `async def _create_worktree(self, name: str, is_overflow: bool = False) -> WorktreeInfo`: path = os.path.join(self.pool_dir, name). Run `git worktree add {path} --detach` from repo_path. Return WorktreeInfo(id=uuid4().hex, path=path, is_overflow=is_overflow).

       `async def _destroy_worktree(self, worktree: WorktreeInfo) -> None`: Run `git worktree remove {worktree.path} --force` from repo_path. Handle errors gracefully with logging.

       `async def _run_git(self, cwd: str, *args: str) -> tuple[str, str, int]`: Use asyncio.create_subprocess_exec("git", *args, cwd=cwd, stdout=PIPE, stderr=PIPE). Await communicate(). Return decoded (stdout, stderr, returncode).

       Properties: `active_count -> int` returning len(self.active), `available_count -> int` returning self.available.qsize().

       `class BranchStrategy`:
       - `def create_branch_name(self, config: BranchConfig) -> str`: Sanitize agent_id: lowercase, replace non-alphanumeric (except hyphen) with hyphen, strip leading/trailing hyphens, collapse multiple hyphens. Format: `f"{config.prefix}/{config.task_id}-{sanitized}"` if task_id non-empty, else `f"{config.prefix}/{sanitized}"`.
       - `async def merge_sequential(self, repo_path: str, source_branch: str, target_branch: str) -> MergeResult`: Run `git checkout {target_branch}`, then `git merge --no-ff {source_branch}`. If returncode 0: return MergeResult(success=True, merged_branch=source_branch, target_branch=target_branch). If returncode != 0: run `git diff --name-only --diff-filter=U` to get conflicted files. Run `git merge --abort` to clean up. Return MergeResult(success=False, merged_branch=source_branch, target_branch=target_branch, conflicts=list_of_files, error="Merge conflicts detected").
       - `async def check_conflicts(self, repo_path: str, source_branch: str, target_branch: str) -> list[str]`: Run `git merge-base {target_branch} {source_branch}` to get common ancestor. Run `git merge-tree {base} {target_branch} {source_branch}`. Parse stdout for lines containing "changed in both" or conflict markers. Return list of conflicted file paths.
       - `async def delete_branch(self, repo_path: str, branch_name: str) -> None`: Run `git branch -d {branch_name}`. On failure, log warning and try `git branch -D {branch_name}`.
