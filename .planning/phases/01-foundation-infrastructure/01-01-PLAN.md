<objective>
Establish the complete monorepo foundation: project scaffolding, Docker development stack (PostgreSQL, Redis, NATS JetStream, Langfuse), database schema with all domain models, shared agent SDK types, and configuration structure.
</objective>

<context>
This is the first phase of CodeBot. Everything else depends on this infrastructure being in place. The monorepo uses uv for Python and pnpm/Turborepo for Node.js.
</context>

<spec>
## 1. Monorepo Scaffolding
- `apps/server/` — FastAPI backend (Python)
- `apps/dashboard/` — React web dashboard (Vite + TypeScript)
- `apps/cli/` — CLI application (TypeScript)
- `libs/agent-sdk/` — Agent base classes and tools (Python)
- `libs/graph-engine/` — Graph execution engine (Python)
- `configs/` — YAML configs for agents, pipelines, providers

## 2. Docker Development Stack
- PostgreSQL 16 (state/config)
- Redis 7 (cache/pubsub)
- NATS 2 with JetStream (event bus)
- Langfuse (LLM observability)

## 3. Database Schema
SQLAlchemy 2.0 models with DeclarativeBase:
- project, agent, task, artifact, event, review, security, checkpoint, experiment, test_result, user (13 models)

## 4. Shared Types
Agent SDK models: agent, enums, events, pipeline, project, task

## 5. Configuration
YAML config directories for agents, pipelines, providers
</spec>
