<objective>
Build the vector store abstraction (LanceDB/Qdrant backends) and Tree-sitter code indexer for structural code understanding.

Purpose: L2 context retrieval and code-aware search depend on a vector store with indexed code symbols. Tree-sitter provides language-aware parsing that regex cannot match (nested structures, docstrings, decorators). These are the "deep search" capabilities that make agents context-aware beyond simple file loading.

Output: vector_store.py with pluggable backends (LanceDB for dev, Qdrant for prod), code_indexer.py with Tree-sitter symbol extraction, and comprehensive tests.
</objective>

       Define `VectorStoreBackend(Protocol)`:
       ```python
       class VectorStoreBackend(Protocol):
           async def upsert(self, id: str, content: str, embedding: list[float],
                           metadata: dict) -> None: ...
           async def query(self, query_embedding: list[float], top_k: int = 5,
                          filter: dict | None = None) -> list[VectorResult]: ...
           async def delete(self, ids: list[str]) -> None: ...
           async def hybrid_search(self, query: str, query_embedding: list[float],
                                  top_k: int = 5,
                                  filter: dict | None = None) -> list[VectorResult]: ...
       ```

       Implement `LanceDBBackend`:
       - `__init__(self, persist_dir: str = "data/lancedb")` -- calls `lancedb.connect(persist_dir)` (sync API wrapped in asyncio.to_thread per research recommendation)
       - Internal table name: `"code_chunks"`
       - Table schema: columns for `id (str)`, `text (str)`, `vector (list[float])`, `file_path (str)`, `symbol_name (str)`, `symbol_kind (str)`, `project_id (str)`, `line_start (int)`, `line_end (int)`
       - `async def upsert(...)`: wrap sync `table.add()` in `asyncio.to_thread()`; if table doesn't exist, create it with first insert
       - `async def query(...)`: wrap sync `table.search(query_embedding).limit(top_k)` in `asyncio.to_thread()`, convert results to VectorResult list
       - `async def delete(...)`: wrap sync `table.delete(f"id IN {ids}")` in `asyncio.to_thread()`
       - `async def hybrid_search(...)`: use sync `table.search(query, query_type="hybrid")` with `RRFReranker()`, wrapped in `asyncio.to_thread()`. Catch errors if FTS index not created and fall back to vector-only search.
       - `async def create_fts_index(self, field: str = "text") -> None`: calls `table.create_fts_index(field, replace=True)` wrapped in asyncio.to_thread()

       Implement `QdrantBackend`:
       - `__init__(self, url: str = "http://localhost:6333", collection_name: str = "codebot")` -- creates `AsyncQdrantClient(url=url)`
       - `async def _ensure_collection(self)`: checks if collection exists, creates with `VectorParams(size=384, distance=Distance.COSINE)` if not
       - `async def upsert(...)`: calls `client.upsert(collection_name, points=[PointStruct(...)])`
       - `async def query(...)`: calls `client.query_points(collection_name, query=embedding, limit=top_k, query_filter=...)`; convert to VectorResult
       - `async def delete(...)`: calls `client.delete(collection_name, points_selector=PointIdsList(points=ids))`
       - `async def hybrid_search(...)`: same as query() for Qdrant (Qdrant's hybrid search requires BM25 index setup which we defer -- fall back to vector search)

    3. Create `apps/server/src/codebot/context/code_indexer.py`:

       ```python
       import tree_sitter_python as tspython
       import tree_sitter_javascript as tsjavascript
       import tree_sitter_typescript as tstypescript
       from tree_sitter import Language, Parser
       ```

       Define language registry:
       ```python
       _LANGUAGES: dict[str, Language] = {
           ".py": Language(tspython.language()),
           ".js": Language(tsjavascript.language()),
           ".ts": Language(tstypescript.language_typescript()),
           ".tsx": Language(tstypescript.language_tsx()),
       }
       ```

       Define query patterns per extension:
       ```python
       _QUERIES: dict[str, dict[str, str]] = {
           ".py": {
               "function": "(function_definition name: (identifier) @name) @definition",
               "class": "(class_definition name: (identifier) @name) @definition",
               "import": "[(import_statement) (import_from_statement)] @import",
               "method": "(class_definition body: (block (function_definition name: (identifier) @name) @method))",
           },
           ".ts": {
               "function": "(function_declaration name: (identifier) @name) @definition",
               "class": "(class_declaration name: (identifier) @name) @definition",
               "import": "(import_statement) @import",
           },
           ".tsx": {
               "function": "(function_declaration name: (identifier) @name) @definition",
               "class": "(class_declaration name: (identifier) @name) @definition",
               "import": "(import_statement) @import",
           },
           ".js": {
               "function": "(function_declaration name: (identifier) @name) @definition",
               "class": "(class_declaration name: (identifier) @name) @definition",
               "import": "(import_statement) @import",
           },
       }
       ```

       Class `CodeIndexer`:
       - `__init__(self, project_root: str | Path)` -- stores project_root as Path
       - `def extract_symbols(self, source: bytes, file_path: str) -> list[CodeSymbol]`:
         - Determine extension from file_path
         - Get Language from _LANGUAGES, return [] if unsupported
         - Create Parser with language, parse source
         - For each query kind in _QUERIES[ext], run query, extract captures
         - For "definition"/"method"/"import" captures (skip "name"-only captures), build CodeSymbol:
           - name: find child node of type "identifier", decode bytes
           - kind: from query key ("function", "class", "method", "import")
           - file_path: the file_path argument
           - line_start: node.start_point.row + 1
           - line_end: node.end_point.row + 1
           - signature: first 200 bytes of node text, decoded
         - Return list of CodeSymbol
       - `async def index_file(self, file_path: str | Path) -> list[CodeSymbol]`:
         - Read file bytes using aiofiles (or asyncio.to_thread with open())
         - Call extract_symbols()
         - Return symbols
       - `async def index_directory(self, directory: str | Path, extensions: list[str] | None = None) -> list[CodeSymbol]`:
         - Walk directory, collect files matching extensions (default: [".py", ".ts", ".tsx", ".js"])
         - Call index_file() for each, aggregate results
         - Return all symbols
