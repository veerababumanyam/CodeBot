<objective>
Build the authentication foundation, standard response envelope, dependency injection layer, and project CRUD endpoints for the CodeBot API server.

Purpose: This plan creates the core API infrastructure that all subsequent endpoints depend on -- JWT/API key authentication, the response envelope contract, FastAPI dependency injection (get_db, get_current_user, require_role), CORS/rate-limit middleware, and the first domain endpoints (auth + projects). Without this foundation, no other API route can be built.

Output: Working auth flow (register/login/refresh/me), project CRUD (create/list/get/update/delete), test suite covering auth and project endpoints, new dependencies installed (pyjwt, bcrypt, slowapi).
